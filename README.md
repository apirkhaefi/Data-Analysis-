# Sleep Health and Lifestyle Data Analysis

## Overview

This project presents an Exploratory Data Analysis (EDA) of the **Sleep Health and Lifestyle Dataset** using Python. The objective is to explore how sleep quality is associated with demographic characteristics, physical activity, stress levels, cardiovascular indicators, BMI, occupation, and sleep disorders.

The project focuses on data cleaning, preprocessing, feature engineering, statistical exploration, and visualization to uncover meaningful patterns within the dataset.

---

# Objectives

* Explore relationships between lifestyle factors and sleep quality.
* Identify factors associated with sleep disorders.
* Perform data preprocessing and feature engineering.
* Create informative visualizations.
* Generate insights from health-related data.

---

# Dataset

Dataset:


* Age

* Sleep Duration
* Quality of Sleep
* Physical Activity Level
* Stress Level

* Blood Pressure
* Heart Rate
* Daily Steps
* Sleep Disorder
